# Consumer Preference for Digital Payment Methods

## Project Overview

This academic research project examines consumer preferences and behavior toward digital payment methods using survey data collected through Google Forms.

## Objectives

* Analyze consumer preferences for digital payment methods.
* Identify factors affecting adoption and satisfaction.
* Apply statistical techniques to derive meaningful insights.

## Data Collection

Primary data was collected using a structured Google Forms survey.

## Tools & Technologies

* Python
* Google Forms
* Pandas
* NumPy
* Matplotlib
* Statistical Analysis

## Techniques Used

* Chi-Square Test of Independence
* Correlation Analysis
* Reliability Analysis (Cronbach's Alpha)
* Data Visualization

## Key Findings

* Consumer preferences vary across payment methods.
* Convenience and usability significantly influence adoption.
* Statistical analysis revealed important relationships among consumer behavior variables.
